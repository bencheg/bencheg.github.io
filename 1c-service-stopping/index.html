Служба Агент сервера 1С:Предприятия 8.3 (x86-64) зависла в состоянии «остановка»

11 ноября 2021 windows, 1C

На самом деле, так можно остановить любую зависшую службу. На примере 1С:Предприятия:

Выясняем pid процесса:

sc queryex "1C:Enterprise 8.3 Server Agent (x86-64)"

Затем:

taskkill /f /pid [PID]

[PID] - ID процесса из команды выше.

Запускаем службу.
